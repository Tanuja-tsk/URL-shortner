# 🔗 URL Shortener App

A simple and efficient **URL Shortener** built using the **MERN Stack (MongoDB, Express, React, Node.js)**.  
This app allows users to convert long URLs into short, shareable links — just like bit.ly!

---

## 🚀 Features

- ✂️ **Shorten Long URLs** instantly  
- 🔁 **Redirect** users to the original link when short URL is visited  
- 🗃️ **MongoDB Integration** for storing and retrieving URLs  
- ⚙️ **Express & Node.js Backend** for handling API requests  
- 💻 **React Frontend** for an easy-to-use interface  
- ✅ **URL Validation** to ensure valid input  
- 📊 (Optional) Click tracking and analytics

---

## 🛠️ Tech Stack

**Frontend:** React.js, Axios, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB (Mongoose)  
**Hosting (Optional):**  
- Frontend → Vercel / Netlify  
- Backend → Render / Railway  
- Database → MongoDB Atlas
